# Delta for smena-negative-stock-resolution

## ADDED Requirements

### Requirement: Отображение отрицательных остатков
Шаг «Анализ отрицательных остатков» ДОЛЖЕН показывать пользователю все позиции с отрицательным конечным остатком за выбранный день по выбранному складу (номенклатура, склад, величина отрицательного остатка), используя существующий механизм анализа остатков товаров на складах. Шаг ограничен отображением отчёта — исполнение исправляющих действий (создание перемещения, инициирование правки рецептуры и т.п.) прямо из мастера НЕ входит в scope этого change (см. `proposal.md → ## Open Questions`).

#### Scenario: Есть отрицательные остатки
- GIVEN за 2026-07-05 по складу «Кухня-1» позиция «Сливки 33%» имеет конечный остаток -2,5 л
- WHEN пользователь открывает шаг «Анализ отрицательных остатков» за 2026-07-05 по «Кухня-1»
- THEN мастер показывает строку с номенклатурой «Сливки 33%», складом «Кухня-1» и величиной остатка -2,5 л

#### Scenario: Явное «всё чисто»
- GIVEN за 2026-07-05 по складу «Кухня-1» нет ни одной позиции с отрицательным конечным остатком
- WHEN пользователь открывает шаг «Анализ отрицательных остатков»
- THEN мастер показывает явное сообщение об отсутствии отрицательных остатков вместо пустой таблицы

### Requirement: Критерий пройденности шага анализа остатков
Шаг «Анализ отрицательных остатков» СЧИТАЕТСЯ пройденным для комбинации день+склад исключительно при полном отсутствии позиций с отрицательным конечным остатком на этом складе за этот день. Исправление остатков выполняется пользователем вне мастера (типовыми документами перемещения/списания/корректировки рецептуры); после исправления пользователь возвращается на этот шаг, чтобы статус пересчитался по актуальным данным.

#### Scenario: Остатки исправлены вне мастера
- GIVEN на предыдущем открытии шага была показана одна позиция с отрицательным остатком по «Кухня-1» за 2026-07-05
- AND пользователь провёл вне мастера документ перемещения, устранивший отрицательный остаток
- WHEN пользователь повторно открывает шаг «Анализ отрицательных остатков» за 2026-07-05 по «Кухня-1»
- THEN мастер показывает отсутствие отрицательных остатков
- AND шаг отмечается как пройденный
