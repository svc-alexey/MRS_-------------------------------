# User Rules

## Контекст проекта

Это **расширение конфигурации** (CFE) `MRS_ЗакрытиеСмены` с префиксом `MRS_`, разрабатываемое поверх **1С:ERP**.

- **Исходники расширения:** текущий репозиторий (`D:\MRS_SMENA`).
- **Основная конфигурация (ERP):** `D:\1c\ERP_XML` — типовые и заимствованные объекты (`КассыККМ`, `Номенклатура` и др.) валидны и описаны там.
- **MCP-серверы:** используют те же эндпоинты, что и проект ERP (`D:\1c\ERP_XML\.cursor\mcp.json`); индексация метаданных и кода относится к основной конфигурации ERP.
- **Политика размещения:** новые объекты — только в расширении (`NEW_OBJECTS_IN=extension` в `.dev.env`); основную конфигурацию не модифицировать без явного указания.

При поиске метаданных, зависимостей и примеров кода через MCP (`codesearch`, `metadatasearch`, `search_metadata`, `execute_metadata_cypher`) ориентироваться на объекты ERP из `D:\1c\ERP_XML`. Код и объекты с префиксом `MRS_` — в текущем репозитории.
